### Common Questions

**Q: How often do I need to disable JWT?**  
A: After EVERY backend code deployment. It auto-enables.

**Q: Why can't I fix this with code?**  
A: JWT verification is a Supabase platform setting, not a code configuration.

**Q: Will this affect security?**  
A: No - your code still handles authentication via middleware. JWT verification just prevents requests from reaching your code.

**Q: Can I automate disabling JWT?**  
A: Not directly. You'd need to use Supabase API (if available) or consider splitting into public/protected Edge Functions.

**Q: How do I know if it's JWT or something else?**  
A: Run `diagnose-ios-connection.html` - it will tell you exactly what's wrong.
